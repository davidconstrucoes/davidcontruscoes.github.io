coloque <h1> no inicio e </h1> no fim de um texpo para fazer do texto um titulo, por exemplo, <h1>título</h1> tornaria em

título

<p>paragrafo</p> cria um paragrafo. tipo este:

paragrafo com proposito de demonstração

coloque <br> para criar uma nova linha deste jeito:
<hr> para fazer uma linha reta como esta
<a href="link">texto</a> cria um link para o site que você colocar entre aspas,
por exemplo <a href="https://google.com">google</a> se tornaria em google
Lembre-se que <h1>título</h1> e <p>paragrafo</p> cria uma nova linha automaticamente
